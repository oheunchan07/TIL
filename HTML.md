# HTML

## HTML이란?
#### 웹페이지를 만드는 코드
- - -
## HTML 문법
* ### TAG
```
<strong></strong>
```
#### `<strong></strong>` : 사이에 있는 텍스트를 강조 표시
<br><br>

```
<u></u>
```
#### `<u></u>` : 사이에 있는 텍스트 아래에 밑줄
<br><br>

```
<h1></h1>
```
#### `<h1></h1>` : 중요도에 따라 뒤에 숫자를 바꿔서 테스트의 크기를 바꿈
<br><br>

```
<br> ,<p></p>
```
#### `<br>` : 줄바꿈 ※무엇인가를 설명하지 않는 태그들은 감싸야 하는 텍스트가 없어 태그를 닫지 않는다
#### `<p></p>` : 사이에 있는 텍스트를 그룹핑 시킴
<br><br>
