# Data Type

## Basic Data Type
> String, Int, Double, Bool, Num  
> ```Num``` 타입은 ```Int```자료형과 ```Double```자료형을 동시에 쓸 수 있다. 

## List
> ```var 변수명 = []``` 처럼 선언한다.  
> 마지막 값 뒤에 ,를 붙히면 정렬된다.  
> 리스트 안에 ```if```나 ```for```를 쓸 수 있다.  

## Map
> ```key-value```로 이루어져있다.  
> ```Map 변수명 = {}```처럼 선언한다.

## Set
> ```Set```은 ```List```랑 비슷하지만 중복된 값을 ```Set```안에 추가해도 해당 ```Set```안에는 변함이 없다
